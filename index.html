01

Главные правила (IT-специфика — читать обязательно)

В IT-офисе главная ценность — техника и документы на столах. Большинство «катастроф» — это залитая клавиатура, выдернутый провод и выброшенная нужная бумажка. Эти правила важнее скорости.

  • Технику не трогаем и не сдвигаем. Компьютеры, мониторы, ноутбуки, серверы, сетевые коробки, провода и удлинители — не отключать, не переставлять, не выдёргивать.
  • Средства НЕ распылять на технику и рядом с розетками. Любой спрей — только на салфетку, в стороне от техники; затем протираем.
  • Воду не лить на пол рядом с проводами и удлинителями. Швабра — хорошо отжатая.
  • Ламинат моем влажной (отжатой), а не мокрой шваброй. Луж быть не должно — от воды разбухают стыки.
  • Бумаги, стикеры, флешки, провода, мелочь на столах — НЕ выбрасываем. Мусор = только очевидный (стаканчики, обёртки, в корзине). Сомневаешься — оставь на месте.
  • Оборудование не выключаем. Если что-то пищит/мигает — это нормально, не трогаем.
  • Окна/розетки/щитки при уборке не трогаем (окна — отдельная услуга по графику).
  • Заметили поломку, протечку, разбитое, странный запах — не убираем сами, сообщаем ответственному.
02

Инвентарь и расходники

  • Швабра-моп с отжимом (флаундер) + ведро с отжимом (лучше двухсекционное)
  • Микрофибра с цветовым кодированием (см. ниже)
  • Распылители (бутылки) под рабочие растворы, подписанные
  • Перчатки (отдельные для санузла и для кухни)
  • Мусорные пакеты разных объёмов
  • Сгон (стяжка) для окон + телескопическая штанга
  • Сухие микрофибры для пыли вокруг техники
  • Запас расходников: туалетная бумага, мыло, бумажные полотенца, губки, средство для посуды

Цветовое кодирование микрофибры

Профстандарт, чтобы не разносить бактерии между зонами. Тряпки разных зон не смешиваем и стираем раздельно.

ЦветГде использовать
КрасныйСанузел (унитаз и зона около него)
СинийОбщие поверхности: столы, подоконники, двери, ресепшен
ЗелёныйКухня: столешница, раковина, техника на кухне
ЖёлтыйРаковины и зеркала в санузле (кроме унитаза)
03

Технология (порядок действий — всегда одинаковый)

  1. Проветрить помещения (если возможно).
  2. Сверху вниз: сначала пыль с высоких поверхностей → столы/поверхности → в последнюю очередь пол.
  3. От дальнего угла к выходу — чтобы не ходить по вымытому полу.
  4. Сначала сухая уборка (пыль, мусор), потом влажная.
  5. Пол — в самом конце, отжатой шваброй, двигаясь к двери.

4.1. Опенспейс и рабочие места0/0

  • Собрать очевидный мусор со столов и пола (стаканчики, обёртки) — бумаги/вещи не трогать
  • Вынести мусор из корзин, заменить пакеты
  • Сортировка мусора: бумага/картон и пластик — в разные пакеты
  • Протереть свободные поверхности столов вокруг техники (спрей — на салфетку)
  • Протереть подоконники, дверные ручки, выключатели (антибактериальной салфеткой)
  • Полить растения (по графику полива — см. примечание)
  • Пыль с открытых поверхностей (полки, тумбы)
  • Вымыть пол отжатой шваброй (ламинат — без луж), от дальнего угла к выходу

синий 4.2. Прихожая / ресепшен0/0

  • Протереть стойку ресепшен, поверхности
  • Привести в порядок входную зону, коврик вытрясти/пропылесосить
  • Зеркало/стеклянные двери — без разводов
  • Вынести мусор из корзины
  • Вымыть пол

зелёный 4.3. Кухонная зона0/0

  • Вымыть посуду / загрузить-разгрузить посудомойку
  • Протереть столешницу, стол, раковину (жироудалитель → смыть)
  • Протереть фасады, микроволновку (внутри/снаружи), кофемашину снаружи
  • Холодильник снаружи; внутри — по графику (см. еженедельный)
  • Вынести мусор, заменить пакет, протереть бак
  • Пополнить: губки, средство для посуды, бумажные полотенца
  • Вымыть пол

красныйжёлтый 4.4. Санузел0/0

  • Унитаз: чистящее + дезинфекция (красный инвентарь)
  • Раковина, смеситель, зеркало (жёлтый инвентарь)
  • Пополнить: туалетная бумага, мыло, бумажные полотенца
  • Вынести мусор
  • Вымыть пол, продезинфицировать
  • Проверить запах (освежитель/нейтрализатор)

4.5. Завершение0/0

  • Проверить, что техника не сдвинута, провода на местах
  • Выключить свет, закрыть окна
  • Инвентарь промыть, тряпки развесить/в стирку
  • Отметить выполнение и оставить комментарии (ниже)

Комментарии / что закончилось / что заметили

05

Еженедельно / раз в 2 недели (добавляется к обычной уборке)

  • Холодильник внутри: убрать просрочку (с предупреждением), протереть полки
  • Плинтусы, ножки столов, нижние поверхности — пыль
  • Двери полностью (полотно + ручки + торцы)
  • Стеклянные перегородки/двери целиком
  • Радиаторы, подоконники тщательно
  • Мягкая мебель/кресла в зоне отдыха — пыль/пятна
  • Чайник/кофемашина — накипь по необходимости
06

Ежемесячно

  • Пыль с верхних поверхностей шкафов, светильников
  • Вентиляционные решётки от пыли
  • Точечная чистка стен/пятен
  • Генеральная кухня (фасады, вытяжка, под техникой)
  • Ревизия расходников и инвентаря, заявка на докупку
07

Мытьё окон — дополнительная услуга (по согласованию, ~1 раз в месяц/квартал)

  • Стёкла изнутри (сгон + средство для стекол, без разводов)
  • Рамы и подоконники
  • Стёкла снаружи (только при безопасном доступе)
  • Москитные сетки (по сезону)

Оплата и периодичность мытья окон — отдельно от основной уборки. Фиксировать дату выполнения.

08

Подбор профессиональной химии

Берём концентраты — дешевле в пересчёте и экономичнее. Главное для вашего пола: pH-нейтральное средство (щёлочь и кислота портят ламинат).

Премиум-B2B

Kiehl, Ecolab, Diversey (TASKI/Suma), Dr.Schnell, Buzil, Tana, Lakma Pro. Лучшее качество, канистры 5–10 л через дилеров. Для маленького офиса избыточно.

Средний проф — оптимум

Pro-Brite, Lakma. Настоящая профхимия, доступно и поштучно. Рекомендуется для вашего офиса.

Бюджет / прозьюмер

Grass, Sanfor Professional. Корни в автохимии, масс-маркет. Рабочее, но нижний тир — не «профстандарт».

⚠ Запрещено

  • Для ламината — средства с аммиаком, кислотой, абразивы, «всё для пола без разбора» (портят покрытие).
  • Не смешивать хлорные средства (санузел) с кислотными — выделяется опасный газ.
  • Спрей для стекол не применять на мониторах — у них своё покрытие (мониторы вообще не трогаем).
  • Все концентраты разводить по инструкции на этикетке — «погуще» не значит «лучше».

Стартовый набор для закупки (на маленький офис)

  • Концентрат для ламината (нейтральный) — 1 л
  • Универсальный спрей + запасной концентрат
  • Средство для стекол
  • Антижир для кухни
  • Средство для посуды + таблетки ПММ
  • Средство для санузла + дезинфекция
  • Антибактериальные салфетки — 1 упаковка
  • Микрофибры 4 цветов, перчатки, пакеты, губки
09

Примечания и контроль (приёмка работ)

  • Полив растений: 1–2 раза в неделю (земля подсохла на 2–3 см → полить; стоит вода в поддоне → не поливать). Уточнить список «капризных».
  • Приёмка: ответственный сотрудник раз в неделю проверяет качество и ставит отметку.
  • Расходники: о заканчивающихся сообщать заранее, не по факту «закончилось».
